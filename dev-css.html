Home

dev.css is a tiny, simple, classless CSS framework inspired by Vercel's Geist design system. It is designed to make any plain HTML file look clean and modern. The minified stylesheet weighs only ~5.5kb and includes both light and dark themes. Scroll down for a demo.

For installation instructions and much more information, see the README.

Who is this for?

dev.css is a great choice for:

  • A simple blog
  • A simple "about me" website
  • Collecting your most used links
  • Prototyping your raw HTML

dev.css was not meant for very complex websites.

If you're using dev.css on your site, feel free to use this "powered by dev.css" badge if you want to promote the project:
